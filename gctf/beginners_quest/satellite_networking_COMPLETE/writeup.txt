1. run get_flag.sh
2. readme.pdf contains satellite name: osmium
3. run init_sat, type in satellite name and press a
4. look at link https://storage.googleapis.com/gctf-2019-attachments/768be4f10429f613eb27fa3e3937fe21c7581bdca97d6909e070ab6f7dbf2fbf
5. get base64 data from it: VXNlcm5hbWU6IHdpcmVzaGFyay1yb2NrcwpQYXNzd29yZDogc3RhcnQtc25pZmZpbmchCg==
6. echo VXNlcm5hbWU6IHdpcmVzaGFyay1yb2NrcwpQYXNzd29yZDogc3RhcnQtc25pZmZpbmchCg== | base64 --decode
7. tcptrack -i enp0s31f6  ->  find out traffic on port 1337
8. listen on this port and save data to 'capture'
   sudo tcpdump -i enp0s31f6 -v -A -w capture port 1337
9. strings capture -> see the solution
